Dynamic Self!


Hello and welcome to Dynamic Self!

Dynamic Self! is a solution to the challenges of effectively showcasing skills through traditional means. This project aims to address these issues by creating a fully custom website built from scratch, offering a more dynamic and personalized way to display skills.

A fully functional contact method for potential collaborators is implemented, with a script managing database queries. Please note that the project is currently hard-coded, not publicly hosted.

Future plans include:

  • Making the portfolio accessible online for wider visibility
  • Using scripts to automate email notifications
  • Interactive project showcasing
